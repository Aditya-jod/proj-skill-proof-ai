How the orchestration fabric works

Every session streams through a decision bus so agents collaborate instantly and administrators see what matters.

Session manager

Creates the workspace, authenticates candidates, and manages the FastAPI WebSocket bridge for real-time updates.

Message bus

All agent messages flow through a central bus, ensuring consistent state snapshots and traceable decision logs.

Adaptive policies

Policies tune difficulty, schedule hints, and trigger integrity escalation based on telemetry emitted by every agent.

Meet the SkillProof agent team

Each agent has a precise job to keep candidates in flow while safeguarding the assessment.

Orchestrator

Routes telemetry, updates session state, and coordinates responses from the rest of the agent network.

Evaluation

Runs compiled code, executes test suites, and publishes rubric-aligned scoring with rationale.

Learning diagnosis

Detects conceptual gaps, recommends new prompts, and justifies each adaptive difficulty change.

Integrity

Monitors focus changes, suspicious paste behavior, and inactivity with escalating review paths.
